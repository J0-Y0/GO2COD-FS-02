About Sotere-market

Store-Market is a scalable eCommerce app built with Django, Django REST Framework, and an SQL database. It features secure APIs for user authentication, product management, and order processing, with a dynamic efficient SQL data handling, JWT authentication and authorization, testing automation, and performance optimization including caching and ORM optimization with database indexing.
